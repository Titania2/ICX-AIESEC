# ICX Portal – AIESEC
Check it out here: https://titania2.github.io/ICX-AIESEC/#/ 

A modern, responsive web platform designed to promote Incoming Exchanges (ICX) for AIESEC. This project focuses on creating a compelling digital experience that connects organizations with global talent while reflecting AIESEC’s mission of cross-cultural exchange and leadership development.

---

## Overview

The ICX Portal is a showcase project centered on:

- Presenting AIESEC’s Incoming Exchange programs in a clear and engaging way  
- Designing an experience that appeals to companies seeking international talent  
- Structuring information to be accessible, intuitive, and user-focused  

This project emphasizes both visual design and functional clarity, treating the platform as a complete user experience rather than just a website.

---

## Features

- Clean, modern interface with strong visual hierarchy  
- Fully responsive design across devices  
- Smooth and fast performance  
- Thoughtful user flow and navigation  
- Scalable and organized code structure

## Design Approach

This project was approached from a UI/UX perspective:

- Focus on clarity over complexity  
- Prioritizing user intent and behavior  
- Creating a balance between branding and usability  
- Ensuring accessibility and readability  

The goal was to build an experience that feels purposeful, not just visually appealing.
